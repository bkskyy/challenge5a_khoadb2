                        IMPORTANT SECURITY NOTES
================================================================================

1. Edit includes/db.php with your database credentials
2. Make uploads directories writable: chmod 755 uploads/
3. Delete setup.php after creating demo accounts
4. Never commit database credentials to Git
5. Use HTTPS in production
6. Change demo password before going live
7. Set file permissions properly

================================================================================
                           TROUBLESHOOTING
================================================================================

Can't login?
→ Run setup.php to create demo accounts
→ Check includes/db.php credentials
→ Verify database was imported

Can't upload files?
→ Check uploads/ directory permissions (755)
→ Verify PHP upload_max_filesize setting
→ Ensure sufficient disk space

Blank white page?
→ Check error_log
→ Enable display_errors in development
→ Verify PHP version is 7.4+

For more help, see INSTALLATION.md
